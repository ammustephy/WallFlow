# WallFlow

A premium wallpaper application built with Flutter.

## Live Backend API
Active Server: [https://wallflow-backend.onrender.com](https://wallflow-backend.onrender.com)

## Features
- **Authentication**: Login, Register, Social Auth (Google).
- **Wallpaper Gallery**: Browse high-quality wallpapers.
- **Profile Management**: Update profile picture and details.
- **Backend**: Node.js + Express + MongoDB (hosted on Render).

## Getting Started

1. **Clone the repository**
2. **Install dependencies**: `flutter pub get`
3. **Run the app**: `flutter run`

For backend setup, refer to the `BACKEND/` directory.
